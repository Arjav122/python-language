# Python Language Notes

A comprehensive collection of Python programming concepts implemented using **Jupyter Notebooks**. This repository is designed for beginners and intermediate learners who want to build a strong foundation in Python through practical examples and explanations.

## 📚 Topics Covered

### Python Fundamentals

* Variables and Data Types
* Boolean Values
* Strings
* Lists
* Tuples
* Dictionaries
* Sets

### Control Flow

* Conditional Statements (`if`, `elif`, `else`)
* `for` Loops
* `while` Loops

### Functions

* Function Definition
* Parameters and Arguments
* Return Values
* Lambda Functions
* Closures
* Decorators

### Object-Oriented Programming (OOP)

* Classes and Objects
* Special (Magic/Dunder) Methods
* Inheritance
* Encapsulation
* Abstraction

### Python Features

* Dictionary Comprehensions
* `enumerate()`
* `zip()`
* `sorted()`

### File Handling

* Reading and Writing Files
* File Modes and File Pointers
* JSON Files
* CSV Files
* Pickle Serialization
